### Project: Code Preview to Image
Code image generator - browser-based rendering

![Preview](docs/output-preview.jpg)


**Stack**

* `Vite` (Build Tool)
* `Shiki` (Syntax Highlighting)
* `html-to-image` (Canvas Rasterization)
* `Tailwind CSS` (Styling)

**Features**

* Text editor with `Live Preview` and Theme Selection
* Background image upload with Blur and Padding options
* Export: `Image` via `Canvas`
